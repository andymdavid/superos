# Bitcoin Platformer PRD (Product Requirements Document)

## 1. Product Overview

### 1.1 Vision
Create an engaging, tongue-in-cheek 2D platformer game in the style of Super Mario that uses Bitcoin as the primary collectible and central banking themes as obstacles and enemies. The game will be playable via web browser, either embedded in a Wix website or hosted on a subdomain.

### 1.2 Target Audience
- Cryptocurrency enthusiasts
- Bitcoin advocates
- Casual gamers
- People interested in financial freedom concepts

### 1.3 Core Game Concept
The player controls a character (based on the provided pixel art sprite) navigating through various levels, collecting Bitcoins while avoiding or defeating central bank-themed obstacles and enemies. The game uses humor and satire to create an entertaining experience while subtly promoting Bitcoin concepts.

## 2. Technical Requirements

### 2.1 Platform
- HTML5 game accessible via web browsers
- Compatible with desktop and mobile devices
- Embeddable in Wix site or hostable on a subdomain

### 2.2 Development Environment
- Cursor AI code editor
- Phaser.js game framework (recommended)
- Asset creation with AI image generation tools

### 2.3 Minimum System Requirements
- Modern web browser with HTML5 support
- JavaScript enabled
- Mobile-responsive design

## 3. Game Mechanics

### 3.1 Player Controls
- Left/Right arrows: Move horizontally
- Spacebar/Up arrow: Jump
- Down arrow: Duck/crouch
- Mobile: Touch-based controls (swipe, tap)

### 3.2 Character Abilities
- Basic movement (walking, running)
- Jumping
- Power-ups (see section 3.6)

### 3.3 Level Structure
- Multiple worlds with different themes
- Each world contains 3-5 levels
- Progressive difficulty
- Final boss battle in each world

### 3.4 Collectibles
- Bitcoins (primary collectible, like Mario coins)
- Satoshis (smaller value collectibles)
- Mining rigs (extra points/power-ups)
- Hardware wallets (save progress)

### 3.5 Obstacles & Enemies
- Janet Yellen boss (throws inflation papers)
- Money printer enemies (shoot fiat bills)
- CBDC agent (tracks and follows player)
- Inflation pits (growing/shrinking platforms)
- Bank vaults (must be broken through)
- FED blocks (moving obstacles)

### 3.6 Power-ups
- "HODL" shield (temporary invincibility)
- "Bull Run" speed boost
- "Hash Power" (attack ability)
- "Cold Storage" (extra life)
- "Lightning Network" (faster movement)

### 3.7 Scoring System
- Bitcoins collected
- Enemies defeated
- Time remaining
- Secrets found

## 4. Game Content & Design

### 4.1 Worlds & Levels
1. **Mining Mountains**
   - Underground caves with mining equipment
   - Bitcoin genesis block as a monument
   - Mining difficulty increases as you progress
   
2. **Wall Street Wasteland**
   - Bank buildings and stock exchanges
   - Money printers as obstacles
   - Corporate suits as enemies
   
3. **Inflation Island**
   - Shrinking platforms representing devaluing currency
   - Rising water/lava level representing inflation
   - Central bank building as the final stage
   
4. **Blockchain Boulevard**
   - Data-themed environment
   - Transaction blocks as platforms
   - Validation checkpoints

### 4.2 Art Style
- Pixel art similar to the provided character sprite
- Retro 16-bit aesthetic
- Vibrant colors for Bitcoin elements
- Muted/grayscale for fiat/banking elements

### 4.3 Sound Design
- 8-bit/chiptune music
- Sound effects for coin collection, jumping, enemy defeat
- Special sounds for power-ups and boss encounters

### 4.4 User Interface
- Minimalist HUD showing:
  - Bitcoin count
  - Lives remaining
  - Current level
  - Score
- Pause menu
- Mobile-friendly controls
